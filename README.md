## Usage

1. **Search for movies**: Enter a movie title in the search bar and click Search
2. **View details**: Click on any movie card to see full details
3. **Add to watchlist**: Click "Add to Watchlist" button on movie details page
4. **Manage watchlist**: Navigate to Watchlist tab to view, filter, and manage saved movies
